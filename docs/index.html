üìã Quick Start

Windows:

1. Download colossus-windows-amd64.exe
2. Open PowerShell and run:
   .\colossus-windows-amd64.exe --help

macOS/Linux:

1. Download the appropriate binary
2. Make it executable:
   chmod +x colossus-*
3. Move to PATH (optional):
   sudo mv colossus-* /usr/local/bin/colossus
4. Run:
   colossus --help

Start the Server:

colossus serve

Download a Model:

colossus models pull tinyllama

Start Chatting:

colossus chat tinyllama
